# Spring-Zerock

> Spring4 MVC Project 연습을 위해 학습한 Zerock 프로젝트입니다.

### 소개
- Spring기반 게시판(파일처리)/댓글 CRUD
- Spring Security를 이용한 암호화처리, 자동로그인
- Spring 웹소켓을 이용한 채팅 기능 구현
